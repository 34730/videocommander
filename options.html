Note

The lower the value of partial loop precision is, the better the precision is when looping partially, and the higher the value is, the worse the precision is. Please remember that when this value is set extremly low, the CPU utilization may go up. The default value (100) is reasonable.

To always show progess bar makes the CPU utilization go up. You can keep showing progress bar temporarily by typing "." (colon) key. It is recommended to leave this option off, and use "." (colon) key when necessary.
